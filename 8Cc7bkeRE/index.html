Cocoapods导入依赖包

  1. 2019-07-05
  2. 276字
  3. 2 min read

1、配置CocoaPods

CocoaPods需要系统内已经安装了ruby，如果没有安装，请先安装。可以使用如下命令：

sudo gem install cocoapods

安装gem工具。随即使用：

pod setup --verbose

之后，只要pod目录没有更新，就尽可能使用这个命令（快得多）：

pod install --verbose --no-repo-update

到此命令执行完毕，cocoapods就可以使用了

2、pod使用

打开终端，进入到项目目录，然后初始化pod

pod init

编辑Podfile文件

source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '10.0'
use_frameworks!
xcodeproj 'SouthSea.xcodeproj'

target 'SouthSea' do
    pod 'Alamofire', '~> 4.7'
    pod 'SwiftyJSON', '~> 4.0'
    pod 'AlamofireImage', '~> 3.5'
	pod 'MJRefresh'
end

安装、下载依赖包

在当前项目目录执行以下命令：

pod install

打开项目

使用workspace文件打开项目

3、常见问题

问题：

Unable to add a source with url `https://github.com/CocoaPods/Specs.git` named

解决方案：

执行以下命令：

pod repo add master https://github.com/CocoaPods/Specs.git

然后提示执行’pod setup’，如果出现

[!] The `master` repo is not a git repo.

进入/users/用户名/.cocoapods/repos，删除master文件夹，然后再次执行

pod setup
