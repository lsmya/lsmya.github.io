OkHttp3使用

  1. 2019-04-25
  2. 383字
  3. 3 min read

  • 添加依赖

implementation 'com.squareup.okhttp3:okhttp:3.11.0'

目前最新的稳定版可能已经不是3.11.0了，可以到官方GitHub来查看最新版本 https://github.com/square/okhttp

okio最新地址：https://github.com/square/okio

  • 异步GET

String url = "http://wwww.baidu.com";
OkHttpClient mOkHttpClient = new OkHttpClient();
Request mRequest = new Request.Builder()
        .url(url)
        .addHeader("key", "value")
        .get()//默认就是GET请求，可以不写
        .build();
mOkHttpClient.newCall(mRequest).enqueue(new Callback() {
    @Override
    public void onFailure(Call call, IOException e) {
    }

    @Override
    public void onResponse(Call call, Response response) throws IOException {
        String data = response.body().string();//返回数据
    }
});

  • 异步POST提交表单

与GET相比，POST多了一个请求体（RequestBody）

String url = "";
OkHttpClient mOkHttpClient = new OkHttpClient();
RequestBody requestBody = new FormBody.Builder()
        .add("key", "value")
        .build();
Request mRequest = new Request.Builder()
        .url(url)
        .post(requestBody)
        .build();

mOkHttpClient.newCall(mRequest).enqueue(new Callback() {
    @Override
    public void onFailure(Call call, IOException e) {
    }

    @Override
    public void onResponse(Call call, Response response) throws IOException {
    }
});

  • 异步POST提交字符串

String url = "";
MediaType mediaType = MediaType.parse("application/json; charset=utf-8");
String requestBody = "I'm String";
Request request = new Request.Builder()
        .url(url)
        .post(RequestBody.create(mediaType, requestBody))
        .build();
OkHttpClient okHttpClient = new OkHttpClient();
okHttpClient.newCall(request).enqueue(new Callback() {
    @Override
    public void onFailure(Call call, IOException e) {
    }

    @Override
    public void onResponse(Call call, Response response) throws IOException {
    }
});

  • POST同时上传文件和参数

OkHttpClient mOkHttpClient = new OkHttpClient();
String url = "";
File file = new File("path");
String TYPE = "application/octet-stream";
RequestBody fileBody = RequestBody.create(MediaType.parse(TYPE), file);
RequestBody requestBody = new MultipartBody.Builder()
        .addFormDataPart("key", "value")
        .setType(MultipartBody.FORM)
        .addFormDataPart("fileKey", file.getName(), fileBody)
        .build();

Request request = new Request.Builder()
        .url(url)
        .addHeader("key", "value")
        .post(requestBody)
        .build();

mOkHttpClient.newCall(request).enqueue(new Callback() {
    @Override
    public void onFailure(Call call, IOException e) {

    }

    @Override
    public void onResponse(Call call, Response response) throws IOException {

    }
});
