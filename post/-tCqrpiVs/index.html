Android使用Lambda表达式

2019-04-25

JDK 8以上

在主工程的build.gradle中的buildTypes下添加

 compileOptions {
            sourceCompatibility JavaVersion.VERSION_1_8
            targetCompatibility JavaVersion.VERSION_1_8
        }

在主工程添加完之后记得在Module里面添加一下，要不然会报错，不知大家有没有遇到

代码混淆

-dontwarn java.lang.invoke.*
-dontwarn **$$Lambda$*

到此就可以使用Lambda表达式了
